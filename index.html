Introduction

The HealthData JQuery UI Plugin, has been developed for front-end management of health-related data.
The plugin aims to

  • Ease creation of form inputs,
  • Improve presentation of data,
  • Perform simple client-side calculations

Requirements

jquery 1.7+
jqueryUI 1.8.16+

Basic Usage

$(document).ready(function() {
  $('#input').healthdata();
});

Where #input is an <input> field.
