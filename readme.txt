README

JQuery UI Widgets for collection of health-related and physiologic data

HEIGHT
	Buttons to choose Metres / Centimeters / Feet+Inches / Inches
	Automagically change input field on change type
	Get value as any of above
	
WEIGHT
	Buttons to choose Kilograms / Stones+Pounds / Pounds
	
BMI
	Get by supplying reference to height + weight elements

BP
	Two input fields styled as systolic / diastolic
	Function to get MAP
	
ALCOHOL UNITS
	Change between units / Pints of beer / Shots of spirit / Glasses of Wine
